# Projeto: Análise de IMC com Python

## Objetivo
Calcular o Índice de Massa Corporal (IMC) a partir de uma base de dados de altura e peso, classificar em categorias e gerar gráficos de distribuição.

## Estrutura dos arquivos
- `IMC_Analise.py`: Script Python com cálculo de IMC, categorização e gráfico.
- `hw_200_com_IMC.csv`: Base de dados original com altura/peso/IMC/categoria.
- `IMC_Distribuicao.png`: Gráfico de distribuição de IMC por categoria.

## Como rodar
1. Certifique-se de ter Python e Pandas instalados.
2. Coloque todos os arquivos na mesma pasta.
3. Execute o script:
   ```bash
   python IMC_Analise.py
